# HealthReminder

一个基于 Qt 的轻量级桌面托盘提醒工具，用于提醒喝水和远眺休息。

## 功能

- 系统托盘常驻运行
- 可自定义喝水提醒时间间隔
- 可自定义远眺休息时间间隔
- 倒计时自动循环提醒
- 居中提示弹窗 + 系统提示音
- 关闭窗口最小化到托盘
- 使用本地 ini 文件存储配置（绿色版，无注册表）

## 构建环境

- Qt 6.x
- MinGW
- Windows 平台
- Qt Creator 编译

## 使用方法

1. 运行 `HealthReminder.exe`
2. 设置喝水和远眺时间间隔
3. 点击「开始」
4. 程序自动最小化到托盘后台运行

单击托盘图标可以恢复主窗口。
